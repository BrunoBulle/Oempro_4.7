# Oempro_4.7
Octeth Oempro 4.7 allows SQL injection.

------------------------------------------

[Additional Information]
I've found a SQL injection in Oempro 4.7. The parameter "CampaignID" in "Campaign.Get" is vulnerable to SQL Injection attacks.
